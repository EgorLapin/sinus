# Sin Array

Задание:
- Заполнить массив значениями sin(x)
- Один период синуса на весь массив
- Размер массива: 10^7
- Посчитать сумму элементов
- Возможность выбора типа массива (`float` или `double`)

## Сборка

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Запуск

После сборки запускаем программу:
```bush
Debug/sin_array.exe
```
При запуске программа предлагает выбрать тип массива:

```bash
Выберите тип массива: f (float) или d (double)
```
